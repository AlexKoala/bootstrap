설치

npm, Composer, Meteor를 통해서 Bootstrap의 Sass 및 JavaScript 소스 파일을 설치하세요.

패키지 관리를 통한 설치에는 문서 또는 전체 빌드 스크립트가 포함되어 있지 않습니다. 또한 우리의 npm 템플릿 저장소를 사용해서 npm으로 Bootstrap 프로젝트를 빠르게 생성할 수 있습니다.

BootstrapCDN

Bootstrap Icons

처음으로 Bootstrap만의 고유한 오픈 소스 SVG 아이콘 라이브러리가 생겼습니다. 컴포넌트 및 문서와 가장 잘 작동하도록 설계되었습니다.

Bootstrap Icons는 Bootstrap 컴포넌트와 가장 잘 작동하도록 설계되었지만 다른 모든 프로젝트에서도 작동합니다. SVG이기 때문에 쉽고 빠르게 크기를 조절할 수 있고, 여러 방법으로 삽입할 수 있고, CSS로 꾸밀 수 있습니다.

공식 테마

Bootstrap 공식 테마 마켓플레이스의 프리미엄 테마를 통해서 Bootstrap를 다음 단계로 올려놓으세요.

테마는 Bootstrap을 기반으로 확장된 고유의 프레임워크, 풍부한 새로운 컴포넌트 및 플러그인, 문서, 강력한 빌드 도구로 만들어졌습니다.
